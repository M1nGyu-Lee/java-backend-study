# Java Backend Study

Java와 Spring Boot를 학습하며 만든 백엔드 학습용 및 포트폴리오 프로젝트 모음입니다.  
이 repo는 학습 기록과 작은 실습 프로젝트를 중심으로 구성되어 있습니다.

## 기술 스택
- Java 17
- Spring Boot 3
- MySQL
- Maven / Gradle
- Git / GitHub

## 폴더 구조
- `java-basics` : Java 기초 문법 및 OOP 연습 코드
- `springboot-study` : Spring Boot 기능 연습 프로젝트
- `project` : 포트폴리오 프로젝트 (로그인, 게시판, 채팅 등)

## 실행 방법
1. 프로젝트 클론

git clone https://github.com/M1ngyu-Lee/java-backend-study.git
cd java-backend-study/project

2. Spring Boot 실행

코드 복사
./mvnw spring-boot:run
